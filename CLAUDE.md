# AI Automation Agency Website

This is a single-page website for an AI automation agency that delivers custom AI solutions to businesses.

## Skill

Always use the `frontend-design` skill when building or editing this site.

## Stack

- Plain HTML, CSS, and Vanilla JS — no frameworks, no build tools
- Raw CSS with custom properties — no Tailwind, no Bootstrap
- Semantic HTML5
- Google Fonts (Inter)
- Inline SVG icons
